# NUMBER GUESSING GAME
## HOW TO RUN
1. Ensure you have Python 3 installed
2. Open your terminal 
3. Navigate to the project folder:
    cd NumberGuessingGame
4. Run the script:
     python3 main.py

## LOGIC FLOW
1. The game imports the random module
2. The computer generates a random integer between 1 and 100
3. The user selects a difficulty level to define the attempts limit
4. A while loop runs as long as attempts > 0 and the number is not guessed
5. User inputs are compared againstthe secret number with conditional logic
